# JASA System

A MERN-based membership and programme management system built for **Jalinan Siswa antara Masyarakat Asli (JASA)**.

## Overview

This system manages:

- Time-based member registration
- Admin approval workflow
- Structured JVC number allocation
- Role-based numbering constraints:
  - MAIN: 1–21
  - SUB: 22–64
  - MEMBER: 65+
- Programme participation management (planned)

## Tech Stack

- MongoDB Atlas
- Express.js
- React (planned)
- Node.js

Backend setup completed:
